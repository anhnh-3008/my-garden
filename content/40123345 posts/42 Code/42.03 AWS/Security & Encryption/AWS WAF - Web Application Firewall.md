## 🌿 What?
- Là dịch vụ cung cấp tường lừa của AWS.
- Tường lửa có tác dụng ngăn ngừa các cuộc tấn công tới trang web của chúng ta thông qua việc lọc các requests HTTP/HTTPS.
- Triển khai trên:
	- ALB
	- API GateWay
	- CloudFront
	- AppSync GraphQL API
	- Cognito User Pool
